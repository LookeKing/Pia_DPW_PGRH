Temas Imagenes Enlaces (hiperenlaces) Propiedades de fuente y texto Propiedad CSS-radio-borde
Introduccion Todas las Paginas Web tienen aun que sea una imagen esto debido que son necesarias para mejorar la presentacion de esta Los hiprtenlaces son enlaces que permiten brincar de una pagina a otra pagina o documento ayudando a la mejor distribucion de las paginas Las propiedades de fuente y texto son muy importantes debido que estas dan mucha mas diversidad a los escritos de la Pagina Web Son las propiedades que definen las esquinas de un elemento ademas de que pueden especificar una imagen para usar de borde
Explicación Para insertar imagenes se necesitara la etiqueta , etiqueta que no tiene una de cierre, ademas de eso se nesecitara del atributo src que indicara la direccion de la imagen que se colocara, imagen que debera estar en la misma carpeta que el codigo El nombre de la pagina a la que llevara el hiperenlace siempre estara entre comillas ademas de que estara despues de la etiqueta a href para posteriormente ponerle como se abrira el hiperenlace e incluso elegir la palabra que sera el hiperenlace hacia la otra pagina Se diferencian en dos Familias de Fuentes ,la familia generica que son un grupo de fuentes de aspecto similar y la familia de fuentes que son fuentes de familia especifica ademas de eso esta la Propiedad Font-family una propiedad que debe contener el nombre de varias fuentes como sistema de respaldo Las Propiedades border radius sirven mucho en la decoracion de una pagina web esto debido que estan diseñadas para usarse como CSS ademas de eso son muy recomendadas por que se puede elegir tamaño, color e incluso utilizar una imagen dentro
